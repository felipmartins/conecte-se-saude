# Requisitos do Sistema

## Objetivos Principais

### Cadastro de Pacientes

#### Detalhamento

- Nome
- CPF
- Data de Nascimento
- Telefone de Contato
- Telefone de Emergência
- Estado civil
- Sexo

### Identificação de Pacientes + Detalhemento de Pacientes

#### Detalhamento

Mostrar as informações do paciente.


### Consulta Médica

#### Detalhamento

Selecionar o paciente e abriria a tela de consulta médica. (basicamente uma tela em com campo de digitação para o médico digitar o que ele fez na consulta) e mandar para imprimir.

### Avaliação Fisioterapeutica

#### Detalhamento

Formulário de anamnese e avaliação fisioterapeutica, seguindo arquivo disponibilizado pela Jordana. + Presença, ausencia, reposição + Tela para visualização de aulas feitas, perdidas ou repostas

### Evolução Diária - Atividade Física

#### Detalhamento

Formulário preenchido diariamente para cada paciente atendido / Com assinatura do fisioterapeuta. (Fazer tela para impressão)


### Pagamento

#### Detalhamento

- Vencimento
- Dia da matrícula
- Valor

### Agenda

#### Detalhamento

Sistema de horários para inserção de pacientes. Presença e avaliação fisioterapeutica.
